# Coffee-Shop-Sales
# ☕ Coffee Shop Sales Analysis Dashboard

## Overview

An interactive Excel dashboard developed to analyze coffee shop sales data and generate actionable business insights. The dashboard tracks sales performance, customer behavior, store performance, and product trends.

## Key Features

* Total Sales, Footfall, Average Bill & Orders per Customer
* Hourly and Day-wise Sales Analysis
* Store-wise Performance Comparison
* Product Category & Size Distribution
* Top 5 Best-Selling Products
* Interactive Month and Day Filters

## Key Insights

* Peak sales occur during morning hours.
* Friday records the highest order volume.
* Hell's Kitchen is the top-performing store.
* Coffee contributes the largest share of revenue.
* Barista Espresso is the best-selling product.

## Tools Used

* Microsoft Excel
* Power Query
* Pivot Tables & Charts
* Slicers
* Dashboard Design

## Outcome

Transformed raw sales data into a visual dashboard that supports data-driven decision-making for sales optimization, inventory planning, and business growth.
